# Add / Remove Friend — DOM Mini Project
A simple and interactive profile card where a single button toggles between Add Friend and Remove Friend states.
Built using HTML, CSS & JavaScript DOM — great for beginners practicing event handling and UI state updates.

## Live Demo


## Features
- Single button friend toggle
- Dynamic button text & color update
- UI state change — Stranger ↔ Friends
- Clean, responsive profile card UI
- Beginner-friendly DOM manipulation

## What You’ll Learn
- DOM selection (querySelector)
- Event listeners (click)
- Updating text & styles dynamically
- Basic UI state management

## Tech Stack
- HTML5
- CSS3
- JavaScript (DOM)
